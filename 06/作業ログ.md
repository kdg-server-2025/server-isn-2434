# server-isn-2434

## 6回目の授業
```bash
# aqua.yamlファイルを作成
# どの授業でも使うためにsever-isn-2434ディレクトリ直下に置く
aqua init

# aquaのツール選択をする
aqua g -i
# aqua g -i を実行するとツールを選択する画面が出てくるので `kind` と入力し `kubernetes-sigs/kind` を選択する
kind

#　追加したツールを使えるようにする
aqua i -l

# aquaのPassを通す
export PATH="$(aqua root-dir)/bin:$PATH"

# 先ほど選択したツールがインストールされていることを確認
kind version

# config を指定して cluster を作成
kind create cluster --config cluster.yaml

# k8s の node 一覧を表示
kubectl get nodes

# Helm のリポジトリを追加
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update

# Helm chart を k8s クラスタに適用
helm install kdg-server-nginx bitnami/nginx

# ポートフォワーディング
kubectl port-forward svc/kdg-server-nginx 8080:80
```
